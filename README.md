<h4>Control Flow Based Student Result System</h4>
<h4>Objective</h4>
1.Design a student result processing system that takes marks as input.<br>
2.Use if-else statement to assign grades.<br>
3.Implement validation logic for marks range.<br>
4.Use loops to allow multiple student entries.<br>
6.Display final result summary with percentage and grade.<br>
